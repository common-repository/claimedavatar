== Description ==
ClaimedAvatar is your OpenID Avatar image. Your avatar will appear beside your name
when you participate (comments or other contents) on OpenID enabled websites.
The ClaimedAvatar Wordpress plugin enables the ClaimedAvatar avatars in Wordpress blogs.

== Installation ==
1. Upload "claimedavatar.php" to the "/wp-content/plugins/" directory
2. Activate the plugin through the "Plugins" menu in WordPress
3. ClaimedAvatar will override the default get_avatar() function with its extended version
